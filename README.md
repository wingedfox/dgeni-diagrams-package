## How to use
### Install
```
npm install dgeni-diagrams-package --save-dev
```

### Configure
```js
var Package = require('dgeni').Package;
new Package('test-package', [ require('dgeni-diagrams-package') ]);
```

That's all, there are no configuration options yet.
